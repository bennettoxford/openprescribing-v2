# openprescribing-v2

This repo contains some experimental code for a new version of [OpenPrescribing](https://openprescribing.net/).

It is deployed to https://beta.openprescribing.net/.

## Measures

This repository includes measures, which are analyses that we want to continue to monitor over time. They are stored as YAML files, in the [measure_definitions](measure_definitions/) directory, using the syntax as described by [schema()](openprescribing/data/measures/measures.py). This is descended from the original [OpenPrescribing measures](https://openprescribing.net/measure/).
